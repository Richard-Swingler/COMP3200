To Run:
1. Install NodeJS and Phonegap command line tools
2. Navigate to index folder (www) in terminal window.
3. Type "phonegap Serve" 
4. Download phonegap development app on your device (Ipad)
5. Type in Ip address displayed in cmd
6. Use application!

A compiled version requires the device to receive development certificate from apple account.
This can be arranged upon request @ rs14g12@soton.ac.uk
This is due to the application not being submitted for appstore approval.
