Following is a breif tutorial of how to use the app!

Create The Room Layout!

Drag from top left to bottom right to create a room of the required dimentions

Note: 50px is equal to 1/2 metre.

Add Room Features!

Drag from the carousel to the desired location

Note: creating window requires to enter the width first!

Add Room Furniture!

Select name then flag

Note: If you select the desk or bed flag it will overide any name

Solve!

Once all furniture has been added click solve!

Add Room Features!

Select from a range of solutions. Delete unwanted solution and recieve new ones!

Get Started!
